<h2 data-start="65" data-end="78">为什么专注越来越难？</h2>
<p data-start="80" data-end="99">在当代社会，我们面临着前所未有的干扰：</p>
<ul data-start="101" data-end="192">
<li data-start="101" data-end="114">
<p data-start="103" data-end="114">社交媒体的即时通知</p>
</li>
<li data-start="115" data-end="130">
<p data-start="117" data-end="130">即时通讯工具的弹窗提醒</p>
</li>
<li data-start="131" data-end="177">
<p data-start="133" data-end="145">多任务处理的工作环境</p>
<ul data-start="148" data-end="177">
<li data-start="148" data-end="162">
<p data-start="150" data-end="162">同时打开十几个标签页</p>
</li>
<li data-start="165" data-end="177">
<p data-start="167" data-end="177">边开会边回复消息</p>
</li>
</ul>
</li>
<li data-start="178" data-end="192">
<p data-start="180" data-end="192">碎片化阅读习惯的形成</p>
</li>
</ul>
<p data-start="194" data-end="241">这些因素共同导致了<strong data-start="203" data-end="216">注意力持续时间缩短</strong>的问题。研究表明，人类的平均专注时长正在逐年下降。</p>
