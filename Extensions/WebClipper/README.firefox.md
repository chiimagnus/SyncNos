# SyncNos WebClipper (Firefox)

Firefox/AMO 相关的构建与分发说明已整合到 `Extensions/WebClipper/AGENTS.md`。

快速入口：

- Firefox 构建：`npm --prefix Extensions/WebClipper run build:firefox`
- AMO 源码包：`npm --prefix Extensions/WebClipper run package:amo-source`

细节与 AMO 约束见：`Extensions/WebClipper/AGENTS.md`、`Extensions/WebClipper/README.amo-source.md`。
